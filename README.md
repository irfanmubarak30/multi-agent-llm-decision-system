# 🤖 Multi-Agent AI Decision System  

A multi-agent AI system that decomposes complex tasks, performs distributed analysis, and generates optimized AI tool recommendations using coordinated LLM agents.

---

## 🧠 System Overview  

This system simulates a **collaborative decision-making pipeline** using multiple specialized AI agents:

- Task decomposition  
- Parallel research and analysis  
- Aggregation and decision synthesis  

---

## ⚙️ Architecture  

### 🧩 Agents  

- **Task Analyzer Agent**  
  Extracts structured requirements from user input  

- **AI Research Agent**  
  Performs tool discovery and comparative analysis  

- **Recommendation Agent**  
  Synthesizes results into an optimized solution  

---

## 🔄 System Flow  

User Input → Task Decomposition → Multi-Agent Processing → Decision Synthesis → Structured Output  

---

## 🎯 Key Capabilities  

- Multi-agent coordination using CrewAI  
- Structured reasoning (JSON outputs)  
- Task-driven decision pipelines  
- Scalable architecture for complex workflows
